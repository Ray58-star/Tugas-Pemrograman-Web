Praktik Memrogram Pada Mata Kuliah Pemrograman Web

Hari rabu tanggal 4 september 2024, saat itu saya memiliki mata kuliah pemrograman web dan akan segera melakukan praktik memrogram. Seminggu sebelum praktik saya dan teman teman kelas telah diberikan sebuah tugas, dan tugas itu adalah memrogram sebuah text code. Text code tersebut dibagi 1 kelas masing-masing per orang. Saya diberikan sebuah text code smaller text html. Sebelum mempraktikkan text code tersebut, saya harus mempelajarinya terlebih dahulu di internet sebelum kemudian mempraktikkannya.

Saya telah mempelajari sebuah text code smaller text html,fungsi dari text code tersebut adalah untuk memperkecil sebuah teks dari teks sekitarnya. Saat praktik kami menunggu giliran dipanggil untuk menampilkan text code kami yang telah dipelajari, dan saat giliran saya dipanggil saya pun mempraktikkan dan menjelaskan text code yang telah saya pelajari. Saya berhasil melakukannya dan dari pengalaman ini saya akhirnya bisa belajar pengetahuan baru dalam memrogram.
